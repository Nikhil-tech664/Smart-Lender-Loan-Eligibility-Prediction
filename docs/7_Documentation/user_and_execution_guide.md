# SmartBridge SDLC Phase 7: Documentation & Executable Guide

## Quick Execution Sequence

To execute the application end-to-end:

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train ML Pipeline & Save Model Pickle
python train_model.py

# 3. Launch Flask Web Server
python app.py
```

Open a web browser and navigate to `http://localhost:5000`.
