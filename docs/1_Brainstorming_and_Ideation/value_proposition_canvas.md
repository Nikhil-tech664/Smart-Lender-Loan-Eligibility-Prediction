# Value Proposition Canvas

## 1. The Customer Profile (Retail Applicant)
* **Customer Jobs:** Submit credit details to secure a loan; understand eligibility metrics.
* **Pains:** Lengthy approval processes, opaque decision criteria, manual DTI calculations, and complex bank forms.
* **Gains:** Instant pre-approvals, clear improvement insights, simple web interface, and accessible history management.

## 2. The Value Map
* **Products & Services:** SmartLender AI Web Application, Flask Predict REST API, and Machine Learning Telemetry Analytics.
* **Pain Relievers:** Instant automated risk scoring (<25 ms), dynamic form validation, clear rejection explanations, and error protection.
* **Gain Creators:** Celebratory feedback animations, interactive gauge charts, telemetry comparison tables, local history search/filters, and multi-format exports.
