# Project Timeline and Milestones

## 1. Project Implementation Timeline
The development of **SmartLender AI** was organized into an intensive 4-week software delivery cycle:

```
Week 1: Requirement Gathering & Architectural Design
======= [Ideation, Empathy Mapping, Wireframing, Stack Selection]

Week 2: ML Pipeline & Backend Development
======= [Synthetic Dataset, Data Imputation, Model Fitting, Flask APIs]

Week 3: Frontend Construction & Core Integration
======= [Glassmorphism CSS, Chart.js Gauge, LocalStorage History, Export tools]

Week 4: Automated Testing, Documentation & Submission
======= [Unittest Assertions, Manual Matrices, Markdown Docs, GitHub Push]
```

---

## 2. Key Milestones

* **M1: Architecture Sign-Off (End of Week 1)**
  - Delivery: Completed empathy map, requirements analysis, and system architecture blueprints.
* **M2: ML Model Serialization (Mid of Week 2)**
  - Delivery: Trained 6 models and saved the best performer (`loan_model.pkl`) alongside visual analytical screenshots.
* **M3: API Core Operational (End of Week 2)**
  - Delivery: Successfully launched Flask endpoints `/predict` and `/model-telemetry`.
* **M4: Mobile Responsive Shell Complete (Mid of Week 3)**
  - Delivery: Frontend form and results card responsive on iOS and Android viewports.
* **M5: Automated Test Sign-Off (Mid of Week 4)**
  - Delivery: 100% test pass rate with unit assertions.
* **M6: GitHub Production Release (End of Week 4)**
  - Delivery: Complete code pushed to remote repository.
